NEditor extension for Dcat-Admin

## 介绍

dcat-neditor 是一款适配于 dcat-admin 的编辑器，基于 neditor 开发的扩展，并集成 135 编辑器，方便用户使用 135 编辑器的模板与海量功能。

## 安装

dcat-admin 版本：>= 2.0

```
composer require laradocs/dcat-neditor
```

## 使用

使用示例

```php
$form->neditor('article');
```
